Turrican (Amiga) — map viewer

drag to pan · scroll or pinch to zoom

These are Turrican's actual playfields, rebuilt straight from the disk image. Each world is streamed off the floppy as a packed block and decompressed by the game's own three-pass decoder (Huffman → LZ77 → RLE) — reimplemented in Go, byte-verified against the emulator. A decoded block is a little directory: a 16-colour palette, a table of 32×32 four-bitplane tiles, and one or more scenes, each a column-major grid of tile indices (the high values are horizontally flipped tiles). The viewer draws every cell as a real tile through the world's palette — so this is the same image the Amiga's blitter assembles, not a screenshot. Pick a scene to jump between the worlds; the first is the opening of World 1, with its floating platforms, the waterfall, the red tower and the “1989” spelled out in the rock. Tick Objects to overlay the scene's object placements — read straight off the disk from the spawner's own lists, each drawn with the first animation frame of the sprite its AI handler installs, at the position the game seeds it. Tick Collision to overlay the solid map — Turrican stores a 4×4 grid of 8×8-pixel solidity bytes per tile, so the mask follows the terrain at sub-tile resolution. The bytes' values, decoded from the player/shot handlers, are coloured: red solid wall, cyan solid that sparks/blocks shots, yellow breakable block, and magenta hazard (drains energy). Each scene opens framed on the Amiga's on-screen view at the player spawn (the green crosshair), the same start the game shows. Drag to pan, scroll to zoom from the whole level to a single tile. See the write-up (Part IV) for how it was decoded. The Music selector plays every TFMX sub-song in the game — the title/menu module in the $1BB00 sound overlay plus each world's own in-game theme module — all rendered to audio by a from-scratch Go reimplementation of Chris Hülsbeck's in-game TFMX player (Part V §7), no emulator in the loop. Songs are named by their module's runtime address and trackstep start.
