# Marble Madness (Amiga) — c/bootscr (boot-screen overlay) annotations.
# Consumed by codetrace68k -annotate; format: ADDR  name  description.
# bootscr is the overlay the launcher LoadSegs and runs (call_seglist) to put up
# the splash and later to dismiss it (Part II §3, Part IV §1). It is a clean C
# program with its full HUNK_SYMBOL table intact, so most names below ARE its own
# symbols; descriptions are added for the key flow. It is built from the EA IFF
# reader (the _Get*ILBM / _*RIFF / _UnPackRow routines), a custom trackdisk-based
# mini-filesystem (_MF* / ReadSecs / _MGetDir — it reads the disk directly rather
# than through AmigaDOS), and graphics.library display setup.
# Relocatable hunk laid out from an arbitrary base $070000.
#

#   adfdump -x /tmp/mm "Marble Madness (Amiga)/Marble_Madness.adf"
